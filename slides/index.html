MODULE 3: Feature detection

  • Safely add new features
  • Polyfills

Example: Geolocation

  • Make location-aware HTML5 apps
  • API: navigator.geolocation.getCurrentPosition
  • Callback: position.coords lat/lon

Modernizr and Polyfills

  • Compatibility tables: caniuse.com
  • Modernizr does feature detection

Build it!

Make your todo app location aware. Be sure to feature detect to ensure that the browser supports geolocation.

  • Feature detection: Modernizr
  • Note: geolocation won't work in file:// URLs
    • python -m SimpleHTTPServer

For bonus points, use yepnope to load a polyfill if there's no geolocation support.
